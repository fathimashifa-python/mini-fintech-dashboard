# Mini FinTech Dashboard

A personal finance tracker built using React.js.

## Features

- Add income and expense transactions
- Edit and delete transactions
- Search and filter transactions
- Dashboard summary
  - Total Income
  - Total Expense
  - Net Balance
  - Top Spending Category
- Pie Chart for expense distribution
- Bar Chart for expense comparison
- Budget recommendation system
- Local Storage persistence

## Tech Stack

- React.js
- Recharts
- JavaScript
- CSS
- Local Storage

## Setup Instructions

1. Clone the repository

git clone https://github.com/fathimashifa-python/mini-fintech-dashboard.git

2. Install dependencies

npm install

3. Start development server

npm start

4. Open

http://localhost:3000
